# BUG-WEB-003: Unsupported File Type Shows Generic Error

| Field | Details |
| --- | --- |
| Module | File Upload |
| Environment | Edge, Windows 11, staging |
| Severity | Low |
| Priority | Medium |
| Status | Open |

## Steps to Reproduce

1. Open the file upload page.
2. Upload an unsupported file type.
3. Review the displayed error message.

## Expected Result

The message explains supported file formats and file size limits.

## Actual Result

A generic "Something went wrong" message is displayed.
