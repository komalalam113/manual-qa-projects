# BUG-MOB-005: Form Fields Overlap After Device Rotation

| Field | Details |
| --- | --- |
| Module | Profile Form |
| Environment | Google Pixel 7, Android 14 |
| Severity | Medium |
| Priority | High |
| Status | Open |

## Steps to Reproduce

1. Open the profile form.
2. Enter text into required fields.
3. Rotate the device from portrait to landscape.

## Expected Result

The form adapts to landscape view and all fields remain readable and editable.

## Actual Result

Several input fields overlap and the Save button is partially hidden.
