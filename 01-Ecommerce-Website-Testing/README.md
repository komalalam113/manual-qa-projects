# E-commerce Website Testing

## Scope

Representative testing for an online shopping experience covering registration, login, product discovery, cart, checkout, payment flow, coupons, wishlist, order history, and responsive behavior.

## QA Activities

- Reviewed core customer journeys and defined test coverage.
- Created functional, regression, smoke, and negative test scenarios.
- Validated checkout behavior across desktop and mobile breakpoints.
- Logged defects with clear reproduction steps, severity, priority, and expected results.
- Summarized execution results and release risks.
