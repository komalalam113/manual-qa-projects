# Test Plan: SaaS Dashboard

## Objective

Validate that core dashboard workflows work correctly for supported user roles and that business-critical data is displayed consistently.

## In Scope

- Authentication and password reset
- Role-based access control
- Dashboard widgets
- Create, read, update, and delete workflows
- Reports and exports
- Notifications
- Profile and account settings
- Session timeout behavior

## Test Types

- Functional testing
- Regression testing
- UAT support
- Exploratory testing
- Negative testing

## Exit Criteria

- All critical user journeys are executed.
- Critical and high defects are resolved or accepted.
- UAT checklist is ready for stakeholder review.
