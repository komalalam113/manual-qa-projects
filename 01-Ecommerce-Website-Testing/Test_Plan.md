# Test Plan: E-commerce Website

## Objective

Verify that major shopping workflows are stable, usable, and ready for release across supported browsers and responsive layouts.

## In Scope

- Account registration and login
- Product search and filters
- Product detail pages
- Cart and wishlist
- Coupon validation
- Checkout and payment handoff
- Order confirmation and order history
- Responsive testing on desktop, tablet, and mobile

## Out of Scope

- Live payment processing
- Supplier inventory integrations
- Production analytics validation

## Test Types

- Functional testing
- Smoke testing
- Regression testing
- Exploratory testing
- Cross-browser testing
- Responsive testing

## Entry Criteria

- Test environment is available.
- Core user flows are deployed.
- Test data is available for registered and guest users.

## Exit Criteria

- Critical and high defects are resolved or accepted by stakeholders.
- Regression test cases are executed.
- Test execution summary is shared with recommendations.
