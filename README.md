# Manual QA Portfolio Projects

This repository showcases representative Manual QA projects based on real-world freelance testing experience. The artifacts are recreated for portfolio use and focus on testing methodology, documentation quality, defect reporting, and end-to-end QA workflows.

Most commercial projects are covered by NDA or platform confidentiality rules, so this repository does not include client names, private URLs, production screenshots, credentials, internal Jira or ClickUp boards, proprietary workflows, or client-owned assets.

## What This Portfolio Demonstrates

- Functional testing
- Regression and smoke testing
- Exploratory testing
- User acceptance testing (UAT)
- Cross-browser and responsive testing
- Mobile app testing
- Accessibility testing
- Manual API validation
- Test planning and test case design
- Bug reporting and test execution summaries
